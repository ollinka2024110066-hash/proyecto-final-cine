# Proyecto Final Cine 🎬

Sistema web de administración de cine desarrollado con PHP, MySQL, HTML, CSS y JavaScript.

## Funcionalidades

- Inicio de sesión de usuarios
- Panel principal (Dashboard)
- CRUD de películas
- CRUD de salas
- CRUD de funciones
- Venta de boletos
- Selección de asientos
- Ticket de compra
- Reporte de ventas

## Tecnologías utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP

## Instalación local

1. Clonar repositorio
2. Copiar carpeta en htdocs de XAMPP
3. Iniciar Apache y MySQL
4. Importar archivo `cine_db.sql` en phpMyAdmin
5. Abrir navegador en:

http://localhost/PROYECTO_FINAL_CINE/
